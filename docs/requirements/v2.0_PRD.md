# AI Learning Portfolio v2.0 PRD - 吉卜力风格视觉升级

## 1. 项目背景
在 v1.0 版本上线后，为了提升用户的浏览体验，建立更强的情感连接，决定对网站进行全面的视觉升级。新的设计语言参考了吉卜力（Ghibli）动画风格，旨在传达自然、治愈、成长的品牌调性。

## 2. 版本目标
*   **视觉重构**：从“活力橙”转向“吉卜力自然风”，营造宁静、治愈的氛围。
*   **情感化设计**：通过色彩和微动效，增强用户与内容的共鸣。
*   **一致性**：确保所有组件（Hero, Timeline, Cards, Footer）遵循新的设计规范。

## 3. 设计规范 (Design System)

### 3.1 色彩系统 (Color Palette)
| 颜色名称 | 色值 (Hex) | 用途 | 语义 |
| :--- | :--- | :--- | :--- |
| **Ghibli Sky** | `#89CFF0` (Default), `#E0F7FA` (Light), `#4FC3F7` (Dark) | 背景渐变、主视觉区 | 天空、自由、梦想 |
| **Ghibli Green** | `#81C784` (Default), `#C8E6C9` (Light), `#2E7D32` (Dark) | 按钮、图标、强调色 | 草地、生长、希望 |
| **Ghibli Cream** | `#FFFDD0` | 卡片背景、辅助色 | 云朵、温暖、柔和 |
| **Ghibli Yellow** | `#FDD835` | 重点标记、高亮提醒 | 阳光、能量、聚焦 |

### 3.2 字体与排版
*   保持原有字体栈，增加文字颜色的柔和度（使用深灰 `#374151` 替代纯黑）。
*   标题与正文对比度需符合 WCAG 标准。

### 3.3 动效规范
*   **Cloud Drift**：背景云朵缓慢漂浮动画（20s linear infinite）。
*   **Float**：头像和关键元素的轻微悬浮效果。
*   **Pulse**：重点按钮和标签的呼吸效果。

## 4. 功能需求 (Functional Requirements)

### 4.1 全局样式 (Global Styles)
*   [x] **背景重构**：替换纯色/旧渐变为 `ghibli-sky` 线性渐变（从天蓝到淡蓝）。
*   [x] **Tailwind 配置**：在 `tailwind.config.js` 中扩展 `colors` 和 `backgroundImage`。

### 4.2 核心组件升级

#### 4.2.1 Hero Section (首页)
*   **背景**：添加半透明云朵装饰元素 (`blur-xl`, `animate-cloud-drift`)。
*   **CTA 按钮**：由橙色改为 `ghibli-green`，悬停加深。
*   **头像**：保留悬浮动画，边框颜色调整为协调的蓝色系。

#### 4.2.2 Journey Section (学习历程)
*   **时间轴节点**：
    *   圆点/图标背景色改为 `ghibli-green`。
    *   连接线改为 `ghibli-green-light`。
*   **卡片样式**：增加 `ghibli-green-light` 边框悬停效果。
*   **标签**：背景改为 `ghibli-cream`，文字为深绿色。

#### 4.2.3 Projects Section (项目展示)
*   **卡片封面**：使用 `ghibli-sky` 到 `ghibli-green` 的自然渐变代替灰色占位。
*   **重点标识**：`Important` 标签使用 `ghibli-yellow`。
*   **按钮组**：主要操作按钮使用 `ghibli-green`。

#### 4.2.4 Reflection Section (心得感悟)
*   **手风琴**：
    *   展开项背景色调整为柔和的白色/淡绿。
    *   文字高亮色从橙色/红色改为深绿/深蓝，减少焦虑感。
*   **建议卡片**：背景使用天空蓝到奶油色的渐变。

#### 4.2.5 Footer & Navigation
*   **导航栏**：文字悬停色改为 `ghibli-green`。
*   **页脚**：社交图标和留言按钮同步新的配色方案。

#### 4.2.6 Loading Screen
*   **加载页**：背景改为天空蓝渐变，进度条使用白色/奶油色。

## 5. 非功能需求 (Non-Functional Requirements)
*   **性能**：CSS 动画需使用 `transform` 和 `opacity` 属性，避免引起重排 (Reflow)。
*   **兼容性**：渐变色需提供回退背景色（Fallback）。
*   **可访问性**：确保白色文字在浅绿色/浅蓝色背景上的对比度（需适当加深背景或文字阴影）。

## 6. 验收标准 (Acceptance Criteria)
*   所有页面不再出现旧版“橙色”主色调。
*   动画流畅无卡顿。
*   视觉风格统一，给人“治愈、清新”的感受。
