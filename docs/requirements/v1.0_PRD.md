## 1. 🎯 项目概述

### 1.1 背景
本项目作为 "Vibe Coding" 课程的毕业设计，旨在构建一个个人 AI 学习成果展示页。通过该网页，展示从零基础入门到能够独立开发 AI 应用的 8 周学习历程。

### 1.2 核心目标
1.  **成果展示**：直观展示 8 周学习路径、关键产出和个人复盘。
2.  **技术实践**：应用 Next.js、Tailwind CSS 等现代前端技术，并跑通 GitHub + Vercel 自动化部署流程。
3.  **个人名片**：打造一张可在线访问、设计精美的个人名片，用于分享和交流。

---

## 2. 📱 功能模块需求

### 2.1 整体设计风格 (Visual Identity)
-   **配色方案**：温暖橙色渐变系（#FFE4B5 到 #FFB347），传递活力与温暖的学习氛围。
-   **设计语言**：现代卡片式设计 (Card UI)，圆角、阴影、微交互。
-   **交互体验**：单页滚动 (Single Page Scroll)，响应式布局，平滑的滚动动画。

### 2.2 核心板块详细需求

#### 2.2.1 🧭 导航栏 (Navigation)
-   **布局**：顶部固定 (Sticky Top)。
-   **样式**：半透明磨砂玻璃效果，滚动时显示阴影。
-   **内容**：
    -   Logo/品牌名：AI Portfolio
    -   链接：首页 | 学习历程 | 项目展示 | 学习复盘 | 联系我
-   **交互**：点击链接平滑滚动到对应锚点区域。

#### 2.2.2 👋 Hero 区域 (首屏)
-   **目标**：第一时间抓住用户注意力，传达身份与状态。
-   **内容**：
    -   **头像**：圆形，带浮动动画与装饰边框。
    -   **主标题**："嘿！我也是个AI学习者"。
    -   **副标题**："正在AI世界里跌跌撞撞"。
    -   **标签**：圆角胶囊样式（AI学习者 | Vibe Coding学员 | DDL战士）。
    -   **CTA 按钮**："查看我的学习历程"（点击下滚）。

#### 2.2.3 🌈 关于我 (About)
-   **布局**：双卡片布局（桌面端并排，移动端竖排）。
-   **内容**：
    -   **卡片 1 (共鸣)**："我们可能有这些共同点" —— 收藏工具党、寻找伙伴、DDL 驱动。
    -   **卡片 2 (特点)**："关于我的一些小特点" —— 复杂变简单、复盘狂魔、分享主义。

#### 2.2.4 ⏳ 学习历程 (Journey / Timeline)
-   **数据结构**：8 周课程节点。
-   **布局**：
    -   桌面端：左右交替分布，中间贯穿虚线。
    -   移动端：左侧垂直对齐。
-   **节点内容**：
    -   **Week 1**：产品思维 & 痛点分析。
    -   **Week 2**：PRD 文档撰写。
    -   **Week 3 (重点)**：首个 AI 应用（知识卡片生成）。
    -   **Week 4**：代码入门 (Gitee 闯关)。
    -   **Week 5**：Vibe Coding 实践。
    -   **Week 6 (里程碑)**：产品上线发布。
    -   **Week 7**：挖掘真实痛点 (DDL 管理)。
    -   **Week 8**：Demo Day 毕业展示。
-   **交互**：滚动触发节点淡入 (Fade-in) 动画，重点项目金色边框高亮。

#### 2.2.5 💻 项目展示 (Projects)
-   **布局**：三栏卡片网格。
-   **展示项目**：
    1.  **AI 知识卡片生成器**：核心 Demo，一键生成精美卡片。
    2.  **情绪日记分析助手**：NLP 实践，情绪可视化。
    3.  **智能周报生成器**：办公提效工具。
-   **交互**：卡片悬停上浮 + 阴影加深 (Hover Effect)。

#### 2.2.6 💡 学习复盘 (Reflection)
-   **形式**：手风琴折叠面板 (Accordion)。
-   **内容**：
    -   **收获 1**：Prompt 工程 —— 模糊到精准的对比。
    -   **收获 2**：产品落地 —— 以前 vs 现在的开发模式对比。
    -   **收获 3**：心路历程 —— 8 周心情曲线（信心-焦虑-突破-享受）。
-   **特色**：支持展开/收起，内容包含列表、对比表、心情色块。

#### 2.2.7 🦶 页脚 (Footer)
-   **内容**：版权信息、社交媒体链接（GitHub, Twitter 等）。
-   **样式**：深色背景，沉浸式收尾。

## 3. 🛠️ 技术栈与非功能需求

### 3.1 技术选型
-   **框架**：Next.js 14 (App Router)
-   **语言**：TypeScript
-   **样式**：Tailwind CSS
-   **部署**：Vercel
-   **版本控制**：Git + GitHub

### 3.2 性能与兼容性
-   **响应式**：完美适配 Mobile (<768px) 和 Desktop (>1024px)。
-   **加载速度**：首屏加载时间 < 1.5s (Loading 骨架屏优化)。
-   **SEO**：基础元数据配置 (Metadata)。
